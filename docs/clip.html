提交到 my-clips 仓库

同光 · 剪藏设置

操作很简单:浏览器选中内容 → 点书签 → 进你自己的 my-clips 仓库 → 每天 8:00 自动上看板。
第一步:填入 GitHub Token

用来把剪藏写进你自己的仓库。Token 只存在你这台浏览器本地,不上传、不进书签。

没有 Token?到 GitHub 生成:Repository access 选 my-clips,Permissions → Contents 设为 Read and write。

第二步:把书签拖到收藏栏

显示收藏栏:Mac ⌘⇧B / Win Ctrl+Shift+B,然后把下面按钮拖上去。

📎 剪到同光 ← 拖我到收藏栏

拖不动?复制代码手动新建书签
第三步:开始剪藏
1任意网页(X/推特会自动认 @作者、微信文章、任何页面)选中要存的文字
2点收藏栏的 「📎 剪到同光」
3弹窗显示「✓ 已剪到同光」即成功(已写进你的 my-clips 仓库)
4每天 8:00 GitHub Actions 自动汇总上看板;想马上看,去仓库 Actions 手动跑一次 构建剪藏索引
