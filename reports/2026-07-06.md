## 🧠 Tier 0 深度参考（扩窗 7 天）

> 这层"每周 1-3 篇深度长文"特征源的近期高价值产出，不算入主篇数。

### K-1. Simon Willison (2026-07-03) "让顶级模型当主管、廉价模型当工人：subagent 模型分层实战"

Willison 提出 agentic coding 的模型分层架构：顶级模型负责判断、审核和综合，实现类工作交给 Sonnet/Haiku 级别的 subagent 执行。实测显示 token 消耗显著降低而代码质量不变。
📌 *给企业的启发：工程团队应立即在 CI/CD 中引入「模型分层策略」——高价模型只审不写、低价模型批量执行——这是 token 成本治理最直接的技术杠杆。*

### K-2. Simon Willison (2026-07-02) "与 coding agent 协作的挑战：大变更集的人机边界"

引用 Geoffrey Litt 在 AIE 大会上的演讲，讨论当 coding agent 构建越来越大的变更集时，人类如何有效审查和协作。Willison 认为当前的 diff 审查工具不适应 agent 时代。
📌 *给企业的启发：DevOps 团队应评估下一代代码审查工具（支持 agent 生成的大 diff 语义摘要），否则「agent 写完没人看得懂」将成为质量黑洞。*

### K-3. Current AI Gap Map v0.1 (2026-07 首周) "开源 AI 生态索引：421 产品深度拆解"

覆盖 266 个软件工具/库、85 个模型、50 个数据集、20 个硬件项目，是目前最全面的开源 AI 生态分类索引。
📌 *给企业的启发：技术选型团队可将 Gap Map 作为 vendor-neutral 的评估起点，特别是在选择 RAG 框架、向量数据库和推理引擎时避免遗漏主流选项。*

### K-4. Fable 5 出口管制 19 天停机复盘 (VentureBeat 2026-07-03) "三分之二的企业已建好对冲方案"

VentureBeat 调查显示，Fable 5 停机期间 67% 的企业客户已有备用模型方案，但 33% 的企业发现自己无意中形成了单一故障点——工作流自动采用最新模型却无回退机制。
📌 *给企业的启发：AI 基础设施必须像云架构一样设计多模型冗余——在采购合同中明确「模型不可用时的 SLA 和自动回退条款」。*

### K-5. BCG AI Radar 2026 (2026-07 首周) "CEO 亲自掌舵 AI 战略，支出翻倍至营收的 1.7%"

调查显示近四分之三的 CEO 亲自担任 AI 首席决策者，四分之五对 AI ROI 比去年更乐观。一线员工 AI 使用率从 51% 飙升至 74%。
📌 *给企业的启发：如果你的 CEO 还没有直接参与 AI 战略决策，你已经落后于 75% 的同行——这不是 CTO 的独角戏，是 C-suite 的必修课。*

### K-6. BCG (2026-07 首周) "Agentic AI 在技术服务领域的 $2000 亿机会"

2026 年标志着从孤立的 agentic 用例到企业级部署的转折点。三分之一的企业已在规模化部署 agentic AI，三分之二期望供应商构建并运营优先用例。
