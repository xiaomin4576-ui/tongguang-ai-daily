**4. ⭐⭐ 🇨🇳 三部委联合发布中国首个 AI Agent 国家政策《智能体规范应用与创新发展实施意见》**

国家网信办、国家发改委、工信部三部门联合印发文件，这是中国首次在国家层面针对 AI Agent 出台专项政策。文件明确 Agent 为「具备自主感知、记忆、决策、交互与执行能力的智能系统」，提出四大原则（安全可控、规范有序、创新驱动、应用牵引），推进 AI Agent 互联协议（AIP）国家标准，19 个典型应用场景写入文件。分级治理：敏感行业备案+检测+召回，低风险领域合规自测+行业自律。随后新浪财经报道 AIP「身份证」国标即将落地。

🔗 来源：国家网信办 + 新浪财经
📌 *这意味着：中国 AI 治理从内容安全正式升级至行为管理，Agent 全生命周期（工具调用、权限管理、供应链安全）将纳入监管视野；AIP 标准将主动对冲 MCP 等私有协议主导中国 Agent 生态的风险。*
