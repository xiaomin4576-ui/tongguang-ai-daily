## 模板字段约束

| 字段 | 约束 |
|---|---|
| 标题 | 中文为主，专有名词保留英文 |
| 摘要 | 80-120 字 |
| "📌 这意味着" | 30-50 字，含具体动作 |
| Top 5 摘要 | 40-60 字 |
| Tier 0 深度参考摘要 | 60-80 字 |
| 质量分 | 5-8/8（低于 5 不收） |
| 每篇必有的字段 | 来源 + 时间 + 质量分 + 链接 |

## 禁止出现的元素清单

- 表格"v5.1 vs v5 vs v4"
- 段落"🎯 跑法说明"
- "本期由 tongguang-ai-daily v5.x 生成"
- "工具调用 ~28 次"
- "新增 / 比上次多挖到"
- "Tier X 抓 Y/Z" 进度日志
- "（v5.1 新增,181 源必含）" 类标注
